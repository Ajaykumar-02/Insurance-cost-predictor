# 🏥 Insurance Cost Prediction by Ajay.

This is a Machine Learning web application that predicts the medical insurance cost of an individual based on various health and demographic factors.

## 🚀 Project Overview
The goal of this project is to provide an accurate estimate of insurance premiums using features like age, BMI, smoking habits, and more. The model is built using Python and deployed as a web app.

## 🛠️ Tech Stack
* **Language:** Python
* **Machine Learning:** Scikit-learn, Pandas, NumPy
* ML Model: Scikit-learn (Linear Regression)
* **Deployment:** Streamlit 
* **Model Storage:** Pickle (.pkl)

## 📋 Features
The model takes the following inputs to predict the cost:
* **Age:** Age of the primary beneficiary.
* **Sex:** Gender (Male/Female).
* **BMI:** Body Mass Index (providing an understanding of body weight).
* **Children:** Number of children covered by health insurance.
* **Smoker:** Smoking habit (Yes/No).
* **Region:** The beneficiary's residential area in the US.
